# Pac-Man Game using OpenGL (FreeGLUT)

A simple Pac-Man game developed in **C++ using OpenGL (FreeGLUT)** for a **Computer Graphics assignment**. The game features an animated Pac-Man, AI-controlled ghosts, score tracking, timer, lives system, pause menu, and win/lose screens.


## Features

* Animated Pac-Man with mouth movement
* 4 AI ghosts with random and chase behavior
* Maze-based gameplay
* Dot collection and score system
* Lives and timer system
* Pause/Resume functionality
* Start menu and game over/win screens
* Keyboard controls using WASD or Arrow Keys


## Technologies Used

* C++
* OpenGL
* FreeGLUT


## Controls

| Key           | Action             |
| ------------- | ------------------ |
| W / ↑         | Move Up            |
| S / ↓         | Move Down          |
| A / ←         | Move Left          |
| D / →         | Move Right         |
| P             | Pause / Resume     |
| ESC           | Open Menu          |
| Enter / Space | Select Menu Option |
| M             | Return to Menu     |


## How to Run

1. Install CodeBlocks with MinGW
2. Add/Open the project in CodeBlocks
3. Make sure FreeGLUT is configured
4. Build and Run the project


## Gameplay

* Eat all dots to win the game
* Avoid ghosts to survive
* Each ghost collision reduces one life
* Ghost speed increases over time
